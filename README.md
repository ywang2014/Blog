###### kafka
消息监听和推送组件，服务解耦解决方案等
###### canal
数据库监听同步组件，跨机房数据库同步解决方案等
###### docker
轻量级的、可移植的、自给自足的容器，web应用自动化打包、发布、部署解决方案

注册中心，服务发现，分布式应用配置管理解决方案

服务注册与发现

动态分布式配置中心，配置共享，服务发现

分布式服务框架，高性能和透明化的RPC远程服务调用方案

跨语言RPC 框架，通过IDL（接口描述语言） 来定义RPC 数据类型和接口
